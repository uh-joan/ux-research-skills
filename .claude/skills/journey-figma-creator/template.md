## Usage Instructions

1. **Replace all bracketed placeholders** with actual content from your research
2. **Add or remove phases** as needed (typical: 4-8 phases)
3. **Include direct quotes** from user interviews for authenticity
4. **Rate emotions** on 1-5 scale for each phase
5. **Be specific** with pain points - these drive insights
6. **List data sources and tools** extracted from user workflow
7. **Save the file** and use the User Journey Mapper skill to generate Figma visualization

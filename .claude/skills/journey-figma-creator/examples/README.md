# Example User Journey Maps

This folder contains example markdown files and their corresponding Figma outputs.

## Available Examples

### 1. Shirley's Forecasting Journey
- **Input:** `../../../DI&A/user-journey-shirley.md`
- **Figma Output:** [Shirley_journey](https://www.figma.com/design/4n4jbHzgnbk6NhfNPF7fME/Shirley_journey)
- **Description:** Annual pharmaceutical product forecasting workflow
- **Phases:** 6 phases from receiving global guidance to tracking performance
- **Key Insights:** Canada-specific data gaps, IQVIA AI usage, data verification needs

## How to Use Examples

1. **View the markdown source:**
   ```bash
   cat ../../../DI&A/user-journey-shirley.md
   ```

2. **Generate from markdown:**
   ```
   Create a journey map from DI&A/user-journey-shirley.md
   ```

3. **View the Figma output:**
   Open the Figma link above to see the generated journey map

## Creating Your Own

1. Copy `../template.md` to start a new journey map
2. Fill in your research data
3. Run the skill command
4. Review and iterate

## Example Command

```
Create a journey map from my-journey.md in https://figma.com/design/YOUR_FILE_KEY
```
